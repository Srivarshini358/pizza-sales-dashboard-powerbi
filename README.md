# Pizza Sales Dashboard - Power BI

## Overview
This project is an interactive Power BI dashboard created to analyze pizza sales performance and generate meaningful business insights using KPIs and visual reports.

## Key Insights
- Total Revenue Analysis
- Daily & Monthly Sales Trends
- Best & Worst Selling Pizzas
- Sales by Pizza Category & Size
- KPI Monitoring and Reporting

## Tools Used
- Power BI
- SQL
- Excel
- Data Cleaning
- Data Visualization
